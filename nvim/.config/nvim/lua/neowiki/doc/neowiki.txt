OVERVIEW                                                     *neowiki-overview*

VimWiki-lite for Neovim. Provides:
  - Daily TODO files with date-stamped paths (YYYY/MM/YYYY-MM-DD.md).
  - Named knowledge bases backed by configurable picker plugins.
  - Markdown helpers: link follow, link insert, table-of-contents, checkbox
    toggle.

==============================================================================
SETUP                                                           *neowiki-setup*

>lua
    require("neowiki").setup({
        wikis = {
            diary    = { path = "~/Notes", diary = true },
            personal = { path = "~/Wiki/personal" },
            work     = { path = "~/Wiki/work" },
        },
        default_wiki = "diary",
        picker = "auto",
        reuse_previous_day = true,
        weekdays_only = false,
    })

COMMANDS                                                     *neowiki-commands*

Diary:
  :WikiToday          Open today's daily file.
  :WikiYesterday      Open yesterday's (last workday if weekdays_only).
  :WikiTomorrow       Open tomorrow's (next workday if weekdays_only).
  :WikiCurrentMonth   Open the current month's folder.

Markdown:
  :WikiCreateLink     Wrap cword/visual as [text](url).
  :WikiCreateIndex    Insert/refresh TOC at top of buffer.
  :WikiFollowLink     Follow link under cursor.
  :WikiToggleCheckBox Toggle - [ ] / - [x] on current line.

Knowledge bases:
  :WikiFind [name]    Find files in named (or default) wiki.
  :WikiGrep [name]    Live grep in named (or default) wiki.
  :WikiOpen [name]    Open the wiki root directory.
  :WikiSelect [act]   vim.ui.select a wiki, then act (find_files|live_grep|open).
